# 🚀 一键部署操作指南

## 前提条件:安装 Git
如果还没安装 Git,访问: https://git-scm.com/download/win
下载后一路 Next 安装即可。

---

## 第 1 步:初始化 Git 仓库
打开 PowerShell,执行:
```powershell
cd "C:\Users\吴传奇\Desktop\new trival-in-OUC"
git init
git add .
git commit -m "Initial commit: 海大旅游网站 + AI 助手"
```

---

## 第 2 步:创建 GitHub 仓库
1. 访问 https://github.com/new
2. Repository name 填: `trival-ouc`
3. 选择 **Private**(私有)
4. 点击 **Create repository**

---

## 第 3 步:推送到 GitHub
创建完仓库后,执行以下命令(替换你的 GitHub 用户名):
```powershell
git remote add origin https://github.com/你的用户名/trival-ouc.git
git branch -M main
git push -u origin main
```

---

## 第 4 步:部署到 Render
1. 访问 https://render.com → 用 GitHub 账号登录
2. 点击 **New +** → **Web Service**
3. 选择 `trival-ouc` 仓库
4. 配置会自动读取 `render.yaml`,无需修改
5. 点击 **Environment** 标签页
6. 添加环境变量:
   - Key: `ALIYUN_API_KEY`
   - Value: `你的阿里云 DashScope API Key（请替换为实际值）`
7. 点击 **Create Web Service**

---

## 第 5 步:等待部署完成
- 约 2-3 分钟
- 部署成功后会显示 URL: `https://trival-ouc-xxxx.onrender.com`
- 访问即可使用网站!

## 💡 常见问题

**Q: 免费版的 Render 会休眠吗?**
A: 会,15 分钟无访问会休眠,下次访问需等待 30 秒唤醒。

**Q: 如何保持网站一直在线?**
A: 可以使用 https://cron-job.org/ 免费服务,每 5 分钟访问一次网站防止休眠。

**Q: API Key 安全吗?**
A: 安全! Key 只存在于 Render 后台环境变量中,代码里看不到。
